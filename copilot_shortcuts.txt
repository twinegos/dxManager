# GitHub Copilot 유용한 단축키

## 기본 제안 관련
- Tab : Copilot 제안 수락
- Esc : 제안 거부/취소
- Ctrl/Cmd + → : 단어별로 부분 수락 (제안의 일부만 받아들이기)
- Alt + ] : 다음 제안 보기
- Alt + [ : 이전 제안 보기

## Copilot Chat 관련 (Copilot Chat 확장팩 있는 경우)
- Ctrl/Cmd + Shift + I : Copilot Chat 열기
- Ctrl/Cmd + I : 인라인 채팅 (커서 위치에서 직접 질문)

## 추가 팁
- 여러 제안이 있을 때 Alt + ] 또는 Alt + [로 선택 가능
- 긴 함수나 클래스 작성 시 Tab으로 단계별 수락 가능
- 주석을 자세히 작성하면 더 정확한 코드 제안을 받을 수 있음

## 작업 흐름 권장사항
1. 함수명이나 클래스명 작성 후 Enter
2. 간단한 주석으로 의도 설명
3. Tab으로 제안 수락하며 단계별 진행
4. Alt + ] 로 다른 구현 방식 확인

## dxManager 프로젝트 특화 팁
- 리팩토링 시 주석에 "# TODO: 이 함수를 별도 모듈로 분리" 같은 의도 명시
- PySide2 관련 코드는 import 문을 먼저 작성하면 더 정확한 제안
- 데이터베이스 관련 코드는 스키마나 컬렉션명을 주석으로 명시
